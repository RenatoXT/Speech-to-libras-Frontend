<!-- GETTING STARTED -->
## Getting Started
### Prerequisites
* npm
* 
```sh
npm install
ng serve
```
<br>

### Installation

1. Clone the repo
```sh
git clone https://github.com/RenatoXT/Speech-to-Libras-Frontend
```
2. Install NPM packages
```sh
npm install
```
3. Run the app
```sh
ng serve
```
1. Enjoy
